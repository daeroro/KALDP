# Linux Kernel 0.01

해당 문서는 오랜 시간 전의 리눅스 커널에 대해 설명하고 있다. 운영체제, 하드웨어에 관심이 있는 독자라면  해당 문서를 통해, 80386 아키텍처에 대해 배울 수 있으며, 기본적인 운영체제 구현에 대해 배울 수 있습니다.

해당 문서의 본 저자들과 나 모두 숙달된 커널 해커는 아니기에 기술적으로 완벽하지 않을 수 있습니다. 따라서 독자 분들의 많은 양해 부탁드립니다.
